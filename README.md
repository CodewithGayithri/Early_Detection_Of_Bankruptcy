# 📉 Early Detection of Bankruptcy using Machine Learning

A predictive analytics project that uses Machine Learning techniques to **identify companies at risk of bankruptcy at an early stage**.
This helps investors, banks, and organizations take preventive actions based on financial data.

---

## 🚀 Features

* 📊 Financial data analysis
* 🤖 Bankruptcy prediction using ML models
* 📈 Data preprocessing & feature selection
* 📉 Model evaluation (Accuracy, Precision, Recall)
* 📁 Clean and structured dataset handling

---

## 🧠 Tech Stack

* **Programming Language:** Python
* **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib
* **Model Used:** Random Forest / Logistic Regression
* **Environment:** Jupyter Notebook / VS Code

---

## 📊 Model Evaluation

* Accuracy
* Precision
* Recall
* F1-Score
---

## 🧠 Learning Outcomes

* Applied Machine Learning for real-world financial problem
* Learned data preprocessing and feature engineering
* Built and evaluated predictive models
* Understood classification techniques
